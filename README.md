
## Tecnologias utilizadas
- Next Js
- React
- Tailwind
- Supabase (Banco Postgre SQL)
- Vercel (Deploying e hospedagem)

## Passos iniciais  

- Certifique de configurar as credencias do banco Supabase no arquivo `.env`
- Execute o servidor de desenvolvimento com : `npm run dev`
- Acesse a aplicação em: http://localhost:3000
